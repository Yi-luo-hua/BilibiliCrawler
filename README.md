# B站爬虫工具 - 评论 / 动态

一个基于Python的Bilibili爬取工具，支持**评论爬取**（视频/动态/专栏文章）与**动态爬取**（用户空间/关注页），提供图形界面操作，数据导出为CSV格式。

使用Cursor,Trae,Warp,antigravity,Claude Code完成。

如果有帮助的话，麻烦点个star⭐️谢谢喵！

## 功能特点

### 评论爬取
- 支持通过视频链接、BV号或AV号爬取评论
- 支持爬取动态评论和专栏文章评论
- 支持爬取主评论和子评论（回复）
- 支持按时间或热度排序

### 动态爬取（v1.30）
- 支持爬取用户空间动态（指定 UID 或空间链接）
- 支持爬取关注页动态流（需扫码登录）
- 扫码登录功能，生成二维码弹窗，B 站 App 扫码即可
- 关键词筛选 + 时间范围过滤（最近1小时~7天）
- 导出动态内容、点赞数、评论数、转发数

### 通用
- 图形界面操作，评论/动态双模式一键切换
- 多线程并发爬取子评论，自适应请求频率
- Light / Dark 双主题切换
- 导出为CSV格式

## 环境要求

- Python 3.8+
- Windows / Linux / macOS

## 安装

```bash
pip install -r requirements.txt
```

## 使用方法

### 直接下载（推荐）

从 [Releases](https://github.com/Yi-luo-hua/BilibiliCommentsCrawler/releases) 页面下载最新版 `exe` 文件，双击运行即可，无需安装 Python 环境。

### 源码运行

```bash
pip install -r requirements.txt
python main.py
```

启动后在图形界面中操作：

**评论爬取**：
1. 选择「评论爬取」模式
2. 输入链接或ID，支持：视频（BV号/链接）、动态（t.bilibili.com/xxx）、专栏（cv号/链接）
3. 设置爬取参数（子评论、最大页数、排序模式等）
4. 选择导出路径，点击"开始爬取"，完成后点击"导出CSV"

**动态爬取**（v1.30）：
1. 选择「动态爬取」模式
2. 点击「扫码登录」，用 B 站 App 扫描二维码
3. 输入目标用户 UID 或空间链接（留空则爬取关注页动态流）
4. 可选：设置关键词筛选、时间范围
5. 点击"开始爬取"，完成后点击"导出CSV"

## 项目结构

```
├── main.py
├── requirements.txt
├── src/
│   ├── api/          # B站API封装（自适应延迟 + 重试 + 扫码登录）
│   ├── crawler/      # 评论爬取 + 动态爬取
│   ├── processor/    # 数据处理和清洗
│   ├── exporter/     # CSV导出（评论 + 动态）
│   └── gui/          # GUI主窗口 + 主题系统 + 控件
├── config/           # 配置文件
└── utils/            # 工具函数
```

## 导出字段

**评论**：评论ID、根评论ID、用户名、用户等级、评论内容、点赞数、回复数、发布时间、IP归属地

**动态**：动态ID、用户名、类型、内容、发布时间、点赞数、评论数、转发数


### Q: 如何切换主题？
A: 点击窗口右上角的 ☀️/🌙 按钮切换亮色/深色主题。
